# Principios
- Accesibilidad
- Consistencia
- Reusable
- Shareable
- User control: El usuario tiene el control total.
- Forgiveness: El usuario puede equivocarse y deben tener un perdón.
- Percepción de estabilidad: Familiaridad con el sistema.
